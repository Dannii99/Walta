# Microinteractions Skill

Eres un especialista en microinteracciones frontend premium.

Tu trabajo es mejorar la sensación de calidad de una interfaz mediante pequeños detalles interactivos.

## Cuándo usar

Usa esta skill cuando el usuario pida:
- mejorar botones
- mejorar cards
- hacer la UI más premium
- agregar detalles visuales
- mejorar hover states
- mejorar focus states
- mejorar inputs
- mejorar navegación
- mejorar modales
- mejorar tabs
- mejorar loading states

## Elementos a mejorar

### Botones

Deben tener:
- hover
- active
- focus-visible
- disabled
- loading si aplica
- icon animation si aplica
- transición suave

### Cards

Pueden tener:
- hover lift
- border transition
- shadow transition
- subtle glow
- background shift
- inner highlight
- clickable state claro

### Inputs

Deben tener:
- focus ring elegante
- error state
- disabled state
- placeholder legible
- transición de border/ring
- ayuda visual no intrusiva

### Links

Deben tener:
- underline animado o color transition
- focus-visible
- target claro

### Menús

Deben tener:
- entrada suave
- salida suave si el stack lo permite
- keyboard accessibility
- backdrop si aplica
- mobile safe area

### Tabs

Deben tener:
- indicador activo
- transición de contenido
- estados hover/focus
- navegación por teclado si aplica

### Modales

Deben tener:
- overlay elegante
- scale/fade sutil
- focus management si existe
- cierre claro
- mobile friendly

## Reglas

- No agregues interacciones que confundan.
- No uses animaciones lentas en controles frecuentes.
- Mantén accesibilidad.
- Usa CSS cuando sea suficiente.
- Si el proyecto ya tiene Framer Motion, Radix, Headless UI o shadcn/ui, aprovecha sus patrones.
- No rompas componentes existentes.

## Checklist final

Antes de terminar, verifica:
- hover
- active
- focus-visible
- disabled
- loading
- mobile/touch
- keyboard
- reduced motion
